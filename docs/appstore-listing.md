## description (4000 chars max)
a real code editor on your phone. not a notes app pretending to be one.

CapyIDE gives you syntax highlighting, a file tree, and AI code assistance - all running locally on your device. no account required, no cloud sync, no telemetry.

EDITOR
- syntax highlighting for kotlin, java, javascript, typescript, python, html, css, json, xml, and markdown
- proper code editor engine (Sora Editor on Android, CodeMirror 6 on iOS)
- create, open, edit, and save files with a project file tree

AI CODE ASSIST
bring your own API key from any of these providers:
- OpenRouter
- Groq
- Azure OpenAI
- Anthropic

ask it to explain code, complete what you're writing, refactor for clarity, or just chat about your project. your code is sent directly to the provider you choose - we never see it.

PRIVACY FIRST
- all files stay on your device
- API keys stored in iOS Keychain
- no analytics, no tracking, no accounts
- ad-supported free tier, or go pro to remove ads

PRO SUBSCRIPTION
- monthly: $3.99/mo
- yearly: $19.99/yr (save ~58%)
- removes all banner ads
- restore purchases anytime in settings

built with kotlin multiplatform and compose multiplatform. open source on github.

## app review notes
this app uses AI features that require the user to provide their own API key from a supported provider (OpenRouter, Groq, Azure OpenAI, or Anthropic). no API keys are included in the app. the AI features will show an error if no key is configured, which is expected behavior. test ad unit IDs are used during review - live ads will display after approval.
