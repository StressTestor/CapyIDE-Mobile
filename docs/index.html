faq

do you store my code?
no. all files stay on your device. nothing is uploaded anywhere.

do you store my API keys?
no. keys are stored in your device's secure storage (iOS Keychain / Android EncryptedSharedPreferences) and sent directly to the AI provider you configured.

how do i restore my subscription?
open settings in the app and tap "restore purchases". if that doesn't work, make sure you're signed into the same Apple ID you used to subscribe.

which AI providers are supported?
OpenRouter, Groq, Azure OpenAI, and Anthropic. bring your own API key.
